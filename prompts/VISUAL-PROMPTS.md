# Visual Image Prompts Library
## Social Strategy — Version 1.0.0
## 20 Industry-Flexible + 3 Generic Creator Prompts

All prompts follow the master structure:
**Subject + Environment + Lighting + Style + Camera + Mood**

Variables in `[BRACKETS]` are replaced by the business profile engine.

**Industries:** Any
**Use case:** Rebranding campaigns, new identity launches

## PROMPT VARIABLES MASTER LIST

| Variable | Description | Example |
|---|---|---|
| `BUSINESS_TYPE` | Type of business | "dental clinic", "retail store" |
| `BUSINESS_NAME` | Brand/business name | "Evergreen Dental" |
| `SERVICE_TYPE` | Type of service | "physiotherapy", "consulting" |
| `PRODUCT_TYPE` | Product being shown | "handmade jewellery" |
| `PRODUCT_OR_SERVICE` | Flexible product or service | "yoga classes" |
| `TARGET_AUDIENCE` | Who the content is for | "mums aged 30-45" |
| `INDUSTRY` | Industry sector | "real estate", "fitness" |
| `BRAND_NAME` | Brand identity name | "Pure Balance Wellness" |
| `CITY` | Location | "Sydney", "Melbourne" |
| `SEASON` | Seasonal context | "Christmas", "Summer" |
| `SEASONAL_ELEMENTS` | Season visual details | "fairy lights, warm candles" |
| `AGE_RANGE` | Character age | "35-45" |
| `GENDER_STYLE` | Presentation style | "professional female" |
| `BRAND_INDUSTRY` | Industry context | "healthcare" |
| `BRAND_CONTEXT` | Brand scenario context | "family insurance brand" |
| `BRAND_OR_PRODUCT` | Flexible brand/product | "morning coffee ritual, branded mug" |
