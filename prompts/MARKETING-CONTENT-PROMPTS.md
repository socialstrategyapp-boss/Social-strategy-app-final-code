# 20 Marketing Content Prompts — Social Strategy
# File: prompts/MARKETING-CONTENT-PROMPTS.md
# Version: 1.0 · Last updated: April 2026
# Usage: These are plug-and-play content creation prompts for the Content Studio.
# Each prompt is industry-flexible and demographic-aware.
# Placeholders: {brandName}, {industry}, {tone}, {targetAudience}, {platform}

---

## HOW TO USE

Replace all {placeholders} with actual values from the brand brief.
Each prompt below maps to a Content Studio "Topic Template" in the UI.

---

## PROMPT 01 — Brand Rebrand / Identity Reveal

```
You are creating a {tone} brand announcement post for {brandName}, a {industry} business.
Target audience: {targetAudience}. Platform: {platform}.

Write a post that:
- Announces a new look, new direction, or evolved brand identity
- Creates excitement and curiosity without giving everything away
- Acknowledges loyal customers and invites them on the journey
- Ends with a strong CTA (follow, visit, or share)
- Includes relevant hashtags for {industry}

The tone must feel {tone}. Make it feel like a genuine moment, not a press release.
```
